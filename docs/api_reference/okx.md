# OKX API 参考

OKX 交易所的 API 接口文档。

## 交易所标识

| 市场 | 标识 | 说明 |

|------|------|------|

| 现货 | `OKX___SPOT` | 加密货币现货交易 |

| 合约 | `OKX___SWAP` | 永续合约 |

| 期权 | `OKX___OPTION` | 期权交易 |

## 认证配置

```python
exchange_kwargs = {
    "OKX___SPOT": {
        "api_key": "your_api_key",
        "secret": "your_secret",
        "passphrase": "your_passphrase",  # 必需
        "testnet": False,
    }
}

```

## API 端点

| 环境 | REST Base URL | WebSocket |

|------|---------------|-----------|

| 生产环境 | `<https://www.okx.com`> | `wss://ws.okx.com:8443/ws/v5/public` |

| 测试环境 | `<https://www.okx.com`> | `wss://wspap.okx.com:8443/ws/v5/public` |

## 特殊参数

### Passphrase

OKX 需要 API Key 的 passphrase，在创建 API Key 时设置。

### 交易对格式

使用中划线格式：`BTC-USDT`

### 请求频率

- 公共接口：20 次/秒
- 私有接口：20 次/秒
- WebSocket 订阅：240 次/秒

## 常用 API

### 获取交易规则

```python
api = api.get_request_api("OKX___SPOT")
instruments = api.get_instruments("SPOT", "BTC-USDT")

```

### 获取账户配置

```python
account_config = api.get_account_config()

```
