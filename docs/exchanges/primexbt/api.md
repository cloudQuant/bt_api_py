# PrimeXBT API 文档

## 文档信息

- 文档版本: 1.0.0
- 创建日期: 2026-03-04
- 官方文档: 有限的公开文档

## 交易所基本信息

- 官方名称: PrimeXBT
- 官网: <https://primexbt.com>
- CMC 衍生品排名: #110
- 交易所类型: CEX (中心化交易所)
- 支持的交易类型: 加密货币合约(Crypto CFD)、外汇(Forex)、商品(Commodities)、股指(Indices)、跟单交易
- 最大杠杆: 200x (加密), 1000x (外汇)
- 手续费: 按交易类型不同

## 认证方式

- PrimeXBT 的公开 API 文档有限
- 主要通过网页和移动端交易
- 部分 API 功能通过合作伙伴/机构渠道提供

## API 状态

> **注意**: PrimeXBT 的 API 接入主要面向机构客户和合作伙伴。
> 零售用户的 API 功能有限。

## 已知功能

### 市场数据
- 获取加密货币价格数据
- 获取外汇/商品/股指数据

### 交易功能
- 开仓/平仓
- 设置止损/止盈
- 查看持仓

### 跟单交易
- 获取策略列表
- 关注/取消关注策略经理

## 特殊说明

- PrimeXBT 是多资产交易平台（不仅限于加密货币）
- 支持加密货币、外汇、商品、股指交易
- 使用 CFD(差价合约)模式
- API 主要面向机构，零售端 API 受限
- 建议优先级: 低（API 文档不完善）
