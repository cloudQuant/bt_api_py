## 交易所基本信息

- 官方名称: BTCC
- 官网: <https://www.btcc.com>
- CMC 衍生品排名: #18
- 交易所类型: CEX (中心化交易所)
- 成立时间: 2011年(最早的加密交易所之一)
- 支持的交易类型: 永续合约(Perpetual Futures)、现货(Spot)
- 最大杠杆: 500x
- 手续费: Maker 0.045%, Taker 0.045% (合约)
