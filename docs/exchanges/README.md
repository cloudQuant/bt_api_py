## 使用说明

每个交易所的API文档包含以下内容：

1. **交易所基本信息** - 官网、API基础URL、支持的交易对
2. **认证方式** - API密钥获取和请求签名方法
3. **市场数据API** - 行情、深度、K线等端点
4. **交易API** - 下单、撤单、查询订单等端点
5. **账户管理API** - 余额查询、资产管理等端点
6. **速率限制** - API调用频率限制规则
7. **WebSocket支持** - 实时数据订阅（如支持）
8. **错误代码** - 常见错误及处理方法
9. **代码示例** - Python示例代码
