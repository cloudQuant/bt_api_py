## 2. 插件契约规范

### 2.1 entry_points group 名

```toml
# 插件包 pyproject.toml
[project.entry-points."bt_api_py.plugins"]
binance = "bt_api_binance.plugin:register_plugin"
```

### 2.2 PluginInfo 数据类

位置：`bt_api_py/plugins/protocol.py`

```python
from dataclasses import dataclass

@dataclass(frozen=True)
class PluginInfo:
    name: str                          # 如 "bt_api_binance"
    version: str                       # 插件自身版本，遵循 semver
    core_requires: str                 # PEP 440 规格，如 ">=2.0,<3.0"
    supported_exchanges: tuple[str, ...]   # 如 ("BINANCE___SPOT",)
    supported_asset_types: tuple[str, ...]  # 如 ("SPOT", "SWAP")
```

### 2.3 插件入口函数签名

```python
def register_plugin(
    registry: "ExchangeRegistry",
    runtime_factory: "GatewayRuntimeRegistrar",
) -> PluginInfo:
    """注册插件能力。

    - 启动时被调用恰好一次
    - 除非真正无法继续，否则不抛异常
    - 必须返回 PluginInfo
    """
```

### 2.4 GatewayRuntimeRegistrar

位置：`bt_api_py/gateway/registrar.py`

```python
class GatewayRuntimeRegistrar:
    _adapters: dict[str, type["BaseGatewayAdapter"]] = {}

    @classmethod
    def register_adapter(cls, exchange_type: str, adapter_cls: type) -> None: ...

    @classmethod
    def get_adapter(cls, exchange_type: str) -> type | None: ...

    @classmethod
    def list_adapters(cls) -> list[str]: ...
```

## 6. CI 检查设计

### 6.1 核心包隔离检查

新增脚本：`scripts/check_core_isolation.py`

```python
#!/usr/bin/env python
"""Fail CI if bt_api_py core imports any bt_api_<exchange> plugin."""
import pathlib, re, sys

FORBIDDEN = re.compile(
    r"^\s*(from|import)\s+bt_api_(binance|okx|ctp|ib_web|mt5)\b",
    re.MULTILINE,
)
CORE_DIR = pathlib.Path("bt_api_py")

failures = []
for py in CORE_DIR.rglob("*.py"):
    text = py.read_text(encoding="utf-8")
    for m in FORBIDDEN.finditer(text):
        failures.append(f"{py}:{m.group().strip()}")

if failures:
    print("Core package must not import plugin packages:")
    for f in failures:
        print("  " + f)
    sys.exit(1)
```

### 6.2 插件冒烟测试

```bash
#!/bin/bash
# scripts/verify_plugin_smoke.sh
set -e

PLUGIN_NAME=$1
VENV=/tmp/plugin_smoke_$$

python -m venv $VENV
source $VENV/bin/activate

pip install -e .
python -c "from bt_api_py.bt_api import BtApi; print('core OK')"

pip install -e packages/${PLUGIN_NAME}
python -c "
from bt_api_py.plugins.loader import PluginLoader
from bt_api_py.registry import ExchangeRegistry
from bt_api_py.gateway.registrar import GatewayRuntimeRegistrar

loader = PluginLoader(ExchangeRegistry, GatewayRuntimeRegistrar)
loader.load_all()
assert '${PLUGIN_NAME}' in loader.loaded
print('plugin smoke OK')
"
```
