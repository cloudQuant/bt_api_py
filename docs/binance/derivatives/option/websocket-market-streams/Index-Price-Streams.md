# Index Price Streams

## Stream Description

Underlying(e.g ETHUSDT) index stream.

**Stream Name:**  
`<symbol>@index`

## Update Speed

**1000ms**

## Response Example
    
    
    {  
        "e":"index",         // event type  
        "E":1661415480351,   // time  
        "s":"ETHUSDT",       // underlying symbol  
        "p":"1707.89008607"  // index price  
    }
