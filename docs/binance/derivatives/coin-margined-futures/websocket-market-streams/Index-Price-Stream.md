# Index Price Stream

## Stream Description

Index Price Stream

## Stream Name

`<pair>@indexPrice` OR `<pair>@indexPrice@1s`

## Update Speed

**3000ms** OR **1000ms**

## Response Example
    
    
      {  
        "e": "indexPriceUpdate",  // Event type  
        "E": 1591261236000,       // Event time  
        "i": "BTCUSD",            // Pair  
        "p": "9636.57860000",     // Index Price  
      }
