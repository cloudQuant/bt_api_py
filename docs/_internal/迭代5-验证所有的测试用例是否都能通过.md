1. 现在项目都已经差不多完成了，现在想要验证下测试用例是否通过

### 任务
1. 写一个脚本跑除了交易所之外的所有的基础测试
2. 写一个脚本跑单个的交易所，需要传入具体的交易所名称，就会跑和这个名称相关的交易所的测试，需要把网络联接相关的测试也都跑一下，可以传入一个参数决定是否跑网络相关的测试
3. 一个交易所一个交易所运行，然后更新readme.md中的交易所的状态。

## 已完成的工作

### 脚本

- **`scripts/run_base_tests.sh`** — 运行所有基础（非网络）测试
  ```bash
  bash scripts/run_base_tests.sh          # 正常运行
  bash scripts/run_base_tests.sh -v       # 详细输出
  bash scripts/run_base_tests.sh -n 4     # 4进程并行
  ```
  覆盖：containers、functions、feeds 单元测试、ctp、stage0/stage1 基础测试；排除 live_* 网络测试。

- **`scripts/run_exchange_tests.sh`** — 运行单个交易所测试
  ```bash
  bash scripts/run_exchange_tests.sh binance           # 仅 mock/单元测试
  bash scripts/run_exchange_tests.sh binance --network # 含网络/live 测试
  bash scripts/run_exchange_tests.sh okx -v
  ```
